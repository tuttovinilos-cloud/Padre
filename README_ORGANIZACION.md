# Estructura Organizada para Revision

## 01_APP_PRINCIPAL
Version principal para revisar flujo real (login, permisos, menu y paginas productivas).

## 02_LABORATORIO
Paginas de pruebas/herramientas separadas del flujo principal.

## 03_RESPALDOS
Respaldo y paginas puente para referencia historica.

## 04_DOCUMENTACION
Notas, guias, SQL y archivos de apoyo.

## Sugerencia de revision
1. Abre primero `01_APP_PRINCIPAL/index.html`.
2. Luego revisa `01_APP_PRINCIPAL/login.html` y permisos.
3. Si todo va bien, pasa a `02_LABORATORIO` y `03_RESPALDOS`.
